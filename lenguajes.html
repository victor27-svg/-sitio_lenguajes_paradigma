Lenguajes Populares

Una selección de lenguajes con sus usos típicos y el/los paradigmas que promueven.

Python

Multiparadigma

Data science, web (Django/Flask), automatización y scripting.

  • Paradigmas: Imperativo, OO, Funcional
  • Ventaja: legibilidad, comunidad enorme

Java

OO · Enterprise

Backends robustos, Android (Kotlin recomendado), ecosistema maduro.

  • Paradigmas: OO
  • Ventaja: portabilidad (JVM)

JavaScript

Event-driven

Web frontend/backend (Node.js), apps híbridas y tiempo real.

  • Paradigmas: Imperativo, Funcional, Reactivo
  • Ventaja: omnipresente en la web

C#

OO

Aplicaciones .NET, Unity (juegos), escritorio y servicios.

  • Paradigmas: OO, Funcional (LINQ)
  • Ventaja: tooling excelente

Go

Concurrente

Servicios cloud, microservicios, CLI. Simplicidad y rendimiento.

  • Paradigmas: Imperativo, Concurrente (goroutines)
  • Ventaja: despliegue fácil

Rust

Seguro

Sistemas, WebAssembly, backends de alto rendimiento.

  • Paradigmas: Imperativo, Funcional
  • Ventaja: seguridad de memoria sin GC

Haskell

Funcional puro

Investigación, compiladores, procesamiento seguro y expresivo.

  • Paradigmas: Funcional
  • Ventaja: expresividad y tipos potentes

SQL

Declarativo

Consultas y gestión de datos. Lenguaje declarativo por excelencia.

  • Paradigmas: Declarativo
  • Ventaja: expresar el "qué" de los datos
